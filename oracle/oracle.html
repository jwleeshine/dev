머리말

IT 자격증 중에 '정보처리기사'와 '컴퓨터활용능력1급'이 있습니다.
두 자격증이 공통으로 포함하는 내용은 '데이터베이스'와 그 데이터베이스를 조작하는 언어인 'SQL(Structured Query Language)'입니다.

또한 'SQL 전문가', 'SQL 개발자'라는 SQL에 특화된 자격증도 있습니다.

자격증이 다는 아니겠지만 데이터를 다루기 위해 사용되는 데이터베이스와 SQL이 중요한 것 같습니다.

데이터베이스는 크게 관계형 데이터베이스와 관계형 데이터베이스가 아닌 것으로 나눌 수 있는데 여기서는 관계형 데이터베이스에 대해서 다루고자 합니다.

관계형 데이터베이스에는 오라클, Microsoft SQL Server, Microsoft Access 등이 있는데 여기서는 '오라클 데이터베이스'에 대해서 다루겠습니다.
